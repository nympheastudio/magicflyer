Si vous avez configuré des modèles d'expédition dans votre compte Seller Central vous pouvez les utiliser dans le module.

Ainsi vous pouvez choisir ici un modèle d'expédition à appliquer pour un profil, un produit ou une déclinaison.

Si vous n'indiquez rien, nulle part, les frais d'expédition configurés par défaut sur Seller Central seront alors appliqués.
